Utils

PC_Full_Version

PC_Full_Version() returns text:

Return a composite version string summarizing pointcloud system components. Includes library, SQL, libxml2, PostgreSQL interface versions, and LAZperf support flag. Useful for debugging or verifying runtime compatibility of the pointcloud extension.

SELECT PC_Full_Version();

POINTCLOUD="1.2.5 2346cc2" PGSQL="170" LIBXML2="2.14.3 LAZPERF enabled=false

PC_Lazperf_Enabled

PC_Lazperf_Enabled() returns boolean:

Return true if the pointcloud extension includes LAZperf compression support.

SELECT PC_Lazperf_Enabled();

t
